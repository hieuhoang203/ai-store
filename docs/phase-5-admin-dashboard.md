# Phase 5 - Admin Dashboard

Current admin app: `fe-ai-store`

Generated capabilities:

- Left menu with Dashboard and all backend-managed entities.
- Dashboard API integration with `/admin/dashboard`.
- Generic CRUD screens from `/admin/entities`.
- Entity list, detail, create, update, delete.
- Pagination and search.
- Toast success/error messages.
- Dark Spotify-inspired UI.

Installed frontend stack additions:

- Axios
- Zustand
- TanStack Query
- React Hook Form
- Zod
- Hookform resolvers
- Recharts
- Date-fns
- clsx
- tailwind-merge

Run:

```bash
cd fe-ai-store
npm run dev
```

URL: http://localhost:2110
